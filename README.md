Ez egy Full-Stack, web alapú jegyzetkezelő applikáció, amely rendelkezik adminisztratív funkciókkal.
Fő funkciói:
- Jegyzetek létrehozása, szerkesztése, törlése, kezelése
- Csoportok létrehozása, kezelése, más felhasználók/tagok kezelése
- Jegyzetek megosztása más felhasználókkal, csoportokkal
A jegyzetek létrehozása felhő alapú (eddig csak a backend host gépén), így a létrehozott jegyzetek megmaradnak ha kilépünk az alkalmazásból
A Frontend React-ben készült, a Backend pedig Node.js-ben.
Ez egy csoportmunka volt a technikumi szakmai vizsgára.
