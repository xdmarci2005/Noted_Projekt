# Noted. Jegyzetkezelő App



Ez egy Online Jegyzetelő-, és kezelő WebApplikáció



## Dokumentáció

[Dokumentáció](https://linktodocumentation)


## Funkciók

- Jegyzetelés alapvető Rich Text szerkeztő funkciókkal
- Saját felhasználó létrehozása, módosítása
- Csoportok:
    - létrehozása
    - csatlakozás
    - szerkeztése
    - tagok kezelése
- Jegyzetek megosztása:
    - felh. -> felhasználó
    - felh. -> csoport
